Crie sua primeira loja Shopify
Antes de começar, você precisa de uma conta Shopify ativa com o tema Horizon instalado.
1. Acesse shopify.com → Start free trial
2. Admin → Online Store → Themes → Visit Theme Store
3. Busque Horizon → Add → Publish
4. Admin → Settings → Apps and sales channels → Develop apps
5. Use a aba Gerar Token neste dashboard para criar o token automaticamente
📦
Prepare seu CSV de produtos
O sistema importa produtos diretamente de um CSV exportado do Shopify. Você pode usar produtos de qualquer loja de referência.
1. Acesse qualquer loja Shopify de referência
2. Admin → Products → Export → All products → Export products
3. Salve o arquivo products_export.csv
4. Na aba Criar Loja, anexe o CSV no campo "Produtos CSV"
✦ Claude vai melhorar todos os títulos e descrições automaticamente
🖼️
Prepare banners para o hero
Banners profissionais fazem a loja parecer legítima. Use IA para gerá-los rapidamente.
1. Na aba Criar Loja, clique em "Ver prompts de imagem"
2. Copie o prompt e gere no Midjourney, DALL-E ou Leonardo AI
3. Tamanho ideal: 1440×800px desktop · 800×1000px mobile
4. Anexe os banners nos campos correspondentes
✦ Sem banner o hero usa cor sólida — funciona mas é menos profissional
✦
