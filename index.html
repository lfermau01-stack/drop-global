Guia Rápido
1 — URL da loja Shopify
Importante: Crie a loja em shopify.com diretamente — não pelo Partners. Assim a loja fica pública sem senha. 3 dias grátis, depois $1/mês.
2 — Token da API Shopify
Como pegar o token
1
No admin da loja vá em Settings → Apps → Develop apps
2
Clique "Create an app" → nome: "drop-global"
3
Clique "Configure Admin API scopes" → ative: write_products, read_products, write_themes, read_themes, write_content, read_content, write_navigations, read_navigations, write_online_store_pages, read_online_store_pages
4
Clique Save → Install app → API credentials → copie o token shpat_...
Atenção: O token aparece apenas uma vez. Copie antes de fechar a página.
3 — Chave da API Anthropic
Como pegar a chave
1
Acesse console.anthropic.com → login com a conta do claude.ai
2
No menu esquerdo clique em "API Keys" → "Create Key"
3
Nome: "drop-global" → copie a chave sk-ant-...
4
Adicione créditos em billing — mínimo $5 (~R$25)
Custo por loja: ~R$5-8 em tokens. Chave fica salva no navegador automaticamente.
