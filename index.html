Guia Rápido v10
Fluxo completo
1
Criar loja Shopify — shopify.com → Start free trial → 3 dias grátis, depois $1/mês por 3 meses.
2
Instalar tema Dawn — Admin → Online Store → Themes → Visit Theme Store → Dawn → Add → Publish.
3
Gerar token Shopify — use a aba "Gerar Token" neste dashboard. Automático.
4
Obter Claude API Key — console.anthropic.com → API Keys → Create Key. Recomendado fortemente.
5
Gerar imagens — clique "Ver prompts" → copie → gere no Midjourney/DALL-E/Nano Banana → anexe.
6
Criar loja — clique "Criar Loja com IA" → aguarde 3-5 minutos → loja pronta.
7
Pagamentos — Admin → Settings → Payments → Shopify Payments ou Stripe.
8
Repetir para mais lojas — Monitor Farm acompanha todas. KV sincroniza em todos os dispositivos.
